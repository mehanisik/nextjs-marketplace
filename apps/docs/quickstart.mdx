# Quickstart Guide

This guide will help you get started with Next Marketplace quickly. We'll walk you through the essential steps to set up your marketplace.

## Prerequisites

Before you begin, make sure you have:

- A modern web browser
- Node.js 18.x or later
- Git installed
- A code editor (VS Code recommended)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-org/next-marketplace.git
cd next-marketplace
```

2. Install dependencies:

```bash
pnpm install
```

3. Set up environment variables:

```bash
cp .env.example .env
```

Edit the `.env` file with your configuration:

```env
# Database
DATABASE_URL="postgresql://user:password@localhost:5432/marketplace"

# Authentication
NEXTAUTH_SECRET="your-secret"
NEXTAUTH_URL="http://localhost:3000"

# API Keys
STRIPE_SECRET_KEY="sk_test_..."
STRIPE_WEBHOOK_SECRET="whsec_..."
```

## Running the Development Environment

1. Start the development server:

```bash
pnpm dev
```

This will start all services:
- Web app: http://localhost:3000
- API: http://localhost:3001
- Admin dashboard: http://localhost:3002
- Creator portal: http://localhost:3003

## Creating Your First Marketplace

1. **Sign Up**
   - Visit http://localhost:3000
   - Click "Get Started"
   - Complete the registration form

2. **Configure Your Marketplace**
   - Set your marketplace name and description
   - Upload your logo and banner
   - Configure payment settings
   - Set up your first product categories

3. **Add Your First Product**
   - Navigate to the Creator Portal
   - Click "Add New Product"
   - Fill in product details
   - Upload product images
   - Set pricing and inventory
   - Publish your product

## Testing the Marketplace

1. **Test User Flow**
   - Create a test buyer account
   - Browse products
   - Add items to cart
   - Complete a test purchase

2. **Test Admin Features**
   - Access the admin dashboard
   - Review orders
   - Manage users
   - View analytics

## Common Issues

### Database Connection
If you encounter database connection issues:
1. Ensure PostgreSQL is running
2. Verify your DATABASE_URL in .env
3. Check database permissions

### Authentication
If you have authentication problems:
1. Verify NEXTAUTH_SECRET is set
2. Check NEXTAUTH_URL matches your environment
3. Clear browser cookies and cache

### Payment Processing
For payment-related issues:
1. Verify Stripe API keys
2. Check webhook configuration
3. Ensure test mode is enabled for development
